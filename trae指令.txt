任务名称：Project V-Tabs - 核心渲染器（完整黄金四轨）

目标：在阶段4的基础上，实现完整的黄金四轨渲染：

使用阶段2的 extractPayload 从 ABC 源码提取音符数据。

在五线谱渲染后，测量每个音符的坐标（系统行底部和音符中心X）。

根据音符的 MIDI 从陶笛字典获取指法状态，调用 drawOcarina12 生成指法图 SVG 节点。

将指法图注入到对应的系统行中，并添加字母谱文本。

处理占位符（skip: true 的不渲染）和字典缺失（静默跳过）。

依赖：确保已安装 abcjs 和 lucide-react（已安装）。阶段2 的 src/components/InstrumentDicts/ocarina12.ts 和 src/lib/abc/ 下的工具函数应该已经存在。

1. 修改 src/components/song/AbcRenderer.tsx，增加注入逻辑
以下是完整的新版 AbcRenderer.tsx 代码。请直接替换原有文件内容。

tsx
'use client';

import { useEffect, useRef, useState } from 'react';
import * as abcjs from 'abcjs';
import { useFontReady } from '@/lib/hooks/useFontReady';
import { useRenderLock } from '@/lib/hooks/useRenderLock';
import Skeleton from '@/components/ui/Skeleton';
import { extractPayload } from '@/lib/abc/parser';
import { injectSpacing } from '@/lib/abc/injectSpacing';
import { DICT, drawOcarina12, MIDI_TO_NAME } from '@/components/InstrumentDicts/ocarina12';
import { NotePayload, NotePosition } from '@/lib/types';

type AbcRendererProps = {
  abcString: string;
  instrumentId: string;
  onRenderStart?: () => void;
  onRenderComplete?: () => void;
};

export default function AbcRenderer({
  abcString,
  instrumentId,
  onRenderStart,
  onRenderComplete,
}: AbcRendererProps) {
  const containerRef = useRef<HTMLDivElement>(null);
  const [hasError, setHasError] = useState(false);
  const [isTimeout, setIsTimeout] = useState(false);
  const fontReady = useFontReady(3000);
  const { startRender, finishRender } = useRenderLock(5000);

  // 注入指法图的函数
  const injectFingering = (svg: SVGSVGElement, payload: NotePayload[]) => {
    // 获取所有系统行
    const systems = svg.querySelectorAll('.abcjs-system');
    if (!systems.length) return;

    const positions: NotePosition[] = [];

    // 第一遍：测量所有音符坐标
    systems.forEach((sys, sysIdx) => {
      const sysBBox = sys.getBBox();
      const systemBottomY = sysBBox.y + sysBBox.height;

      const notes = sys.querySelectorAll('.abcjs-note');
      notes.forEach(note => {
        const noteBBox = note.getBBox();
        const centerX = noteBBox.x + noteBBox.width / 2;
        positions.push({
          systemIndex: sysIdx,
          x: centerX,
          systemBottomY,
        });
      });
    });

    // 第二遍：遍历 payload 和 positions（索引对齐），注入指法图
    payload.forEach((item, idx) => {
      if (item.skip) return; // 占位符不渲染
      const pos = positions[idx];
      if (!pos) {
        console.warn(`位置缺失，索引 ${idx}`);
        return;
      }

      const state = DICT[item.midi];
      if (!state) {
        // 静默跳过（生产环境不报错），可打印调试信息
        if (process.env.NODE_ENV === 'development') {
          console.warn(`字典缺失 MIDI: ${item.midi}`);
        }
        return;
      }

      // 生成指法图
      const { element: fingerG, width, height } = drawOcarina12(state);

      // 计算偏移量：指法图中心约在 (3000, 2500)，顶部约 500，底部约 4500
      // 我们希望指法图的顶部（y≈500）对齐到 systemBottomY
      const tx = pos.x - 3000;       // 水平对齐中心
      const ty = pos.systemBottomY - 500; // 垂直对齐顶部

      fingerG.setAttribute('transform', `translate(${tx}, ${ty})`);

      // 找到对应的系统行
      const targetSys = systems[pos.systemIndex];
      targetSys.appendChild(fingerG);

      // 添加字母谱文本
      const text = document.createElementNS('http://www.w3.org/2000/svg', 'text');
      const name = MIDI_TO_NAME[item.midi];
      if (name) {
        text.textContent = `${name.letter}${name.octave}`;
      } else {
        text.textContent = '?';
      }
      text.setAttribute('x', pos.x.toString());
      text.setAttribute('y', (pos.systemBottomY + 30).toString()); // 放在指法图下方
      text.setAttribute('font-size', '20');
      text.setAttribute('fill', '#3E2723');
      text.setAttribute('text-anchor', 'middle');
      text.setAttribute('font-weight', 'bold');
      targetSys.appendChild(text);
    });
  };

  // 降级处理
  const handleFallback = () => {
    setIsTimeout(true);
    if (containerRef.current) {
      containerRef.current.innerHTML = ''; // 清空以防残留
    }
    onRenderComplete?.();
  };

  useEffect(() => {
    if (!fontReady) return;

    setHasError(false);
    setIsTimeout(false);
    onRenderStart?.();

    // 启动渲染锁
    startRender(handleFallback);

    // 渲染五线谱
    if (containerRef.current && abcString) {
      try {
        // 1. 先进行间距注入
        const modifiedAbc = injectSpacing(abcString, instrumentId);

        // 2. 清空容器
        containerRef.current.innerHTML = '';

        // 3. 渲染五线谱
        abcjs.renderAbc(containerRef.current, modifiedAbc, {
          responsive: 'resize',
          add_classes: true,
          staffwidth: 600,
          paddingtop: 20,
          paddingbottom: 30,
        });

        // 4. 提取 payload
        const payload = extractPayload(abcString); // 注意：用原始 ABC 提取，不是修改后的

        // 5. 获取 SVG 元素
        const svg = containerRef.current.querySelector('svg');
        if (!svg) {
          throw new Error('SVG not found');
        }

        // 6. 注入指法图
        injectFingering(svg, payload);

        // 7. 结束锁
        finishRender();
        onRenderComplete?.();
      } catch (err) {
        console.error('渲染错误:', err);
        setHasError(true);
        finishRender();
        onRenderComplete?.();
      }
    }

    return () => {
      finishRender();
      if (containerRef.current) {
        containerRef.current.innerHTML = '';
      }
    };
  }, [abcString, fontReady, instrumentId, startRender, finishRender, onRenderStart, onRenderComplete]);

  if (!fontReady) {
    return <Skeleton />;
  }

  if (hasError) {
    return (
      <div className="p-8 text-center text-wood-dark bg-red-50 rounded-lg">
        ❌ 乐谱加载失败，请稍后重试。
      </div>
    );
  }

  return (
    <div className="relative">
      {isTimeout && (
        <div className="absolute top-2 right-2 bg-yellow-100 text-yellow-800 text-xs px-2 py-1 rounded">
          加载超时，已显示基础谱面
        </div>
      )}
      <div ref={containerRef} className="abc-paper"></div>
    </div>
  );
}
2. 修复可能的问题：extractPayload 中的 parseOnly 导入
请确保 src/lib/abc/parser.ts 中正确导入 abcjs 的 parseOnly 函数。如果之前未导入，请修改文件头部：

typescript
import { parseOnly } from 'abcjs';
如果 parseOnly 不存在于顶层导出，可能需要使用 abcjs.parseOnly。但根据文档，abcjs 默认导出包含 parseOnly。为了安全，可以改为：

typescript
import * as abcjs from 'abcjs';
const { parseOnly } = abcjs;
请在 parser.ts 中检查并修正。如果没问题则跳过。

4. 可能的注意事项
坐标偏移量微调：tx = pos.x - 3000 和 ty = pos.systemBottomY - 500 是基于陶笛原始 SVG 尺寸的经验值。如果指法图位置偏左或偏右，可能需要微调。可以在注入后通过浏览器检查指法图位置，如果偏移，可以调整这些常数（例如改为 -2800 或 -3200）。但先按此值，有问题再微调。

系统行识别：abcjs 生成的系统行类名 .abcjs-system 是稳定的，但如果未来版本变化，需注意。

字母谱字体：使用系统默认 serif 字体，已在 Tailwind 中配置。
