# Competitor Distribution Research

Concise summary of public competitor/channel observations for melody tabs, letter notes, and beginner instrument content.

## Scope

This note uses publicly visible evidence only. It does not claim paid-tool backlink counts or domain metrics.

## Main Takeaway

Pinterest is a good fit for this niche because the content is visual, song-specific, and beginner-friendly. Treat Pinterest as a discovery and click channel that feeds canonical song / learn pages, not as a replacement for SEO or outreach.

Best near-term pattern:

1. make one vertical image per selected song or intent
2. show a partial preview, not the whole useful asset
3. send traffic to the canonical song / learn page
4. use strong internal links to extend the session

## Public Patterns Observed

Noob Notes:

- visible Pinterest presence
- beginner-first positioning
- song-specific visual discovery
- worth borrowing: partial previews, vertical song creatives, clear beginner promise
- do not copy: unclear-rights background art or heavy community features before traffic exists

Letter Note Player:

- visible YouTube / lesson ecosystem
- shows a multi-channel model rather than pure search
- worth borrowing later: song pages as destinations for external media traffic
- not first move: full YouTube production before image workflow is stable

OcarinaSongbook / RiffSpot:

- preview-first and conversion-oriented structure
- networked music-site relationship
- worth borrowing: public page must be more useful than off-site preview
- not first move: hard paywall / PDF-first model

Community signals:

- ocarina users publicly mention Pinterest, Instagram, Reddit, and community sources when looking for tabs
- visual/social discovery is natural for this niche

## Channel Priority

1. Pinterest: cheapest repeatable visual channel.
2. Reddit / community answering: selective, only when the page directly answers the question.
3. Teacher / beginner resource outreach: slower but better for durable citations.
4. YouTube / Shorts: promising but higher production cost.

## Link-Earning Assets To Consider

Future assets that are more citation-worthy than generic song pages:

- printable beginner fingering charts
- classroom starter-song collections
- first-performance resource pages
- instrument-specific cheat sheets
- public-domain beginner resource pages for teachers/parents
